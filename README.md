# WRO 2026 Future Engineers Robot

 Team Project

This repository contains the code and documentation for our autonomous robot built for the WRO 2026 Future Engineers competition.

 Robot Description

Our robot is a fully autonomous vehicle designed to navigate the competition field using sensors and an Arduino microcontroller.

 Hardware Components

- Arduino board
- Motor driver (H-Bridge)
- DC Motors
- Servo Motor
- Ultrasonic Sensors
- Battery
- Chassis

 Software

Programming language:
- Arduino C/C++

 Features

- Autonomous movement
- Obstacle detection
- Automatic steering control
- Sensor-based navigation
